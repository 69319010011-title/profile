## 📚 รายวิชา: Backend Development

เนื้อหาในรายวิชานี้ประกอบไปด้วยการเรียนรู้โครงสร้างและเครื่องมือที่สำคัญสำหรับการพัฒนา Backend ดังนี้:

### 1. Introduction to Node.js & Runtime Environment
* ทำความเข้าใจสถาปัตยกรรมแบบ Event-Driven และ Non-blocking I/O
* การติดตั้งและการใช้งาน Node.js และ Package Manager (NPM)

### 2. RESTful API Development
* การจัดการ HTTP Methods (GET, POST, PUT, DELETE)
* การใช้งาน Routing และ Middleware
* การจัดการ Request และ Response ในรูปแบบ JSON

### 3. Database Connection
* การเชื่อมต่อและจัดการฐานข้อมูล (SQL / NoSQL)
* การใช้งาน ORM/ODM เพื่อจัดการข้อมูล
